Me he desempeñado en campos como data science con proyectos de analisis de datos con python y tkinter asi mismo cuento con experiencia en el area de programacion front end .

La función de un programador front-end es desarrollar y mantener la interfaz de usuario de una aplicación web. Su objetivo principal es crear una experiencia visual atractiva y funcional para los usuarios, asegurándose de que la aplicación sea fácil de usar, estéticamente agradable y compatible con múltiples dispositivos y navegadores.

Bulma

Bulma es un marco CSS de código abierto, simple y moderno , que depende del módulo flexbox (se utiliza para desarrollar una estructura de diseño receptiva).

Tailwind

Tailwind es realizar un reinicio completo de estilos, de esta forma evitando las típicas inconsistencias que encontramos a la hora de estilar.

Proyectos

Calculadora

Crear una calculadora es un ejercicio común y útil para aquellos que están aprendiendo a programar. Ayuda a familiarizarse con conceptos básicos como la entrada y salida de datos, operaciones matemáticas, control de flujo y estructuras de datos.

Chat de Intercom

El propósito de crear un diagrama de flujo en un chat de Intercom, puede ser proporcionar una representación visual clara y concisa del flujo de interacción entre el sistema de chat y los usuarios.

Proyecto con animaciones

Entretenimiento y experiencia del usuario: Las animaciones pueden hacer que un proyecto sea más atractivo y agradable para los usuarios. Pueden agregar elementos visuales dinámicos, interactividad y efectos especiales que hacen que la experiencia del usuario sea más atractiva y emocionante.
